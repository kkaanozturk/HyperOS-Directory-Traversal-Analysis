# CVE-2025-2844: HyperOS Theme Manager Directory Traversal

Bu repository, HyperOS Theme Manager bileşenindeki (simüle edilmiş) Directory Traversal (Dizin Atlama) zafiyeti (CVE-2025-2844) için kavram kanıtı (Proof of Concept - PoC) ve zafiyet analizini içermektedir.

## 📝 Proje İçeriği

- `/simulated_server`: Zafiyeti barındıran örnek HyperOS Theme Manager simülasyonu (Flask).
- `/poc_python`: Zafiyeti sömürmek için yazılmış ilk Python PoC aracı.
- `/poc_rust`: Hız ve boyut optimizasyonu sağlanmış, Rust ile yeniden yazılmış PoC aracı.
- `/docs`: Zafiyetin teorik altyapısı, etki analizi ve çözüm önerileri (Mitigation).
